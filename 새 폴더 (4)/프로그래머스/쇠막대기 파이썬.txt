(나오면 스택에 넣고 )나오면 스택 팝하고
막대기 길이 2~5이런느낌으로 잡고(캐릭터하나당 1)
(스택에 왼쪽꺼위치넣고 뺄때 위치 아니까 그거두개 배열로 작성)

 레이저 좌표준다음에 (레이저는 길이 1이면 레이저로 판단 그냥 왼쪽수로 넣으면댈듯)
 막대기 사이좌표에잇는 레이저수당 +1 기본 +1(막대기 하나당 루프돌리고 레이저 루프돌려서 이중루프로 확인)
 
 
	아라로 루프돌려서
		왼쪽나오면 현재위치 스택푸시
		오른쪽나오면 스택팝해서 왼쪽오른쪽 적기
	루프끝나면 
	루프돌려서 
		오른쪽에서 왼쪽빼서 1인애들 레이저 리스트로 모으고
		아닌애들 막대기 리스트로 모으고
	루프끝나면
	막대기로 루프돌리고(i)
		레이저로 루프돌려서(j)
			사이 레이저 수 +1만큼 리턴값에 +
			
	리턴
	
	
def solution(arra):
    answer = 0
    lo=0
    stack=[]
    laz=[]
    stick=[]
    dic=[]
    for p in arra: #아라로 루프돌려서
        if p=='(':  #왼쪽나오면 
            stack.append(lo)#현재위치 스택푸시
        else :      #오른쪽나오면 
              dic.append([stack.pop(),lo])  #스택팝해서 왼쪽오른쪽 적기
        lo+=1#위치+1
    
    for l,r in dic: #루프돌려서 
        if (r-l)==1:   #오른쪽에서 왼쪽빼서 1인애들 레이저 리스트로 모으고
            laz.append(l)
        else:           #아닌애들 막대기 리스트로 모으고
            stick.append([l,r])
    for l,r in stick:#막대기로 루프돌리고(i)
        for z in laz:#레이저로 루프돌려서(j)

            if l<z<r:
                answer+=1 #사이 레이저 수 +1만큼 리턴값에 +
               # print(l,' ',z,' ',r)
              #  print(l,'  ',r)
        answer+=1  #처음 막대기+1
    #print(stick)
    #print(laz)
    return answer
